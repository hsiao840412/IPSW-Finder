# 📱 IPSW-Finder

---

**IPSW-Finder** 是一個輕量級的工具，旨在幫助使用者快速、輕鬆地尋找和下載 Apple 裝置（iPhone、iPad 和 Mac）的 **IPSW** 檔案。
* **目標：** 提供一個簡單且可靠的方式來獲取特定裝置和版本所需的 IPSW 檔案。
* **用途：** 適用於需要先下載 IPSW 檔案服務客戶的門市或維修中心，過濾連結後複製並可貼至下載器一次下載。
* **原理：** 使用 https://ipsw.me 的 API 過濾連結。

##  安裝與使用 

1.  前往 [Releases](https://github.com/hsiao840412/IPSW-Finder/releases) 頁面下載最新版本的 `IPSW Finder.dmg`。
2.  開啟後將 App 拖入「應用程式」資料夾。
